# 팀 개발 배분 — 프로그래머 3명 · 디자이너 2명

> 이 문서는 **업무 분장의 기준 문서**입니다.  
> 각 구현 문서(02~10) 상단의 담당 표기는 이 문서의 분장 기준을 따릅니다.
