## 1. 보스 스프라이트 제작 이론

### 보스 크기 규격

보스는 일반 적보다 **크게** 만들어야 존재감이 느껴집니다.

| 대상 | 크기 (픽셀) | 게임 내 크기 (PPU 32 기준) |
|------|-------------|---------------------------|
| 플레이어 | 32×48 | 약 1×1.5유닛 |
| 일반 적 | 32×32 | 약 1×1유닛 |
| **보스** | 96×96 ~ 128×128 | 약 3×3 ~ 4×4유닛 |

보스를 크게 만들면 플레이어가 "위협적이다"는 느낌을 받습니다.  
너무 크면 화면을 가리므로 3×3유닛 내외가 적당합니다.

### 보스 디자인 방향 (기획서 기준)

기획서의 보스는 **어두운 저주 구체** 컨셉입니다.

| 요소 | 가이드 |
|------|--------|
| 색상 | 검은색, 짙은 보라 |
| 형태 | 원형 또는 불규칙 구체 |
| 포인트 | 눈처럼 보이는 빛나는 코어 |
| 분위기 | 어둡고 불길한 느낌 |

스프라이트 제작이 어려우면 **흰색 원 스프라이트를 플레이스홀더**로 쓰고 나중에 교체합니다.

### 스프라이트 임포트 설정

| 항목 | 값 |
|------|-----|
| Texture Type | Sprite (2D and UI) |
| Pixels Per Unit | 32 |
| Mesh Type | Full Rect |
| Filter Mode | Point (no filter) |

## 2. 탄막 프리팹 만들기

### 탄막(BossProjectile) 이론

탄막은 "Instantiate → 일정 방향으로 이동 → 플레이어에게 닿으면 데미지 → 자동 삭제"하는 오브젝트입니다.

투사체가 너무 많이 쌓이면 성능이 나빠지므로 `AutoDestroy`로 자동 삭제합니다.

### 탄막 스프라이트 규격

| 항목 | 값 |
|------|-----|
| 크기 | 16×16 px |
| 색상 | 보라/검정 계열 |
| 형태 | 작은 원 또는 구체 |
| Pixels Per Unit | 32 |

### 탄막 프리팹 제작 순서

1. Hierarchy → **Create Empty** → 이름 `BossProjectile`.
2. **SpriteRenderer** 추가 → 탄막 스프라이트 할당.
3. **CircleCollider2D** 추가 → **Is Trigger** 체크.
4. **Rigidbody2D** 추가 → Body Type: **Kinematic** (코드로 이동 처리).
5. `BossProjectile.cs` 추가 (09-코드에서 만들 예정).
6. 프리팹 저장: `Assets/Prefabs/Boss/BossProjectile.prefab`.

## 3. 강타 이펙트 프리팹 (선택)

강타 발동 시 바닥에 표시되는 경고/충격파 이펙트입니다.

1. Hierarchy → **Create Empty** → `Effect_BossSlam`.
2. 자식 → `SpriteRenderer` + 원형 또는 물결 스프라이트 (붉은색).
3. 루트에 `AutoDestroy.cs` → lifetime: 0.4초.
4. 프리팹 저장: `Assets/Prefabs/Boss/Effect_BossSlam.prefab`.

## 4. BossData ScriptableObject 에셋 만들기

### 전제 조건

`BossData.cs` 스크립트가 있어야 에셋을 만들 수 있습니다.  
09-코드 문서에서 스크립트를 만들고 돌아오세요.

### 에셋 생성

1. `Assets/Data/Boss/` 폴더 만들기.
2. 우클릭 → **Create → Game → Boss Data**.
3. 이름: `BossData_Final`.

### 에셋 채우기

| 필드 | 값 |
|------|-----|
| Boss Id | Boss_Final |
| Max Hp | 200 |
| Contact Damage | 15 |
| Projectile Prefab | BossProjectile 프리팹 연결 |

**Phases 배열 3개 설정:**

## 5. 보스 씬 기본 구성

`Stage_Boss` 씬을 만들고 아래를 배치합니다.

| 오브젝트 | 설명 |
|----------|------|
| `Grid` / `Tilemap` | 넓은 바닥 타일 (아레나형) |
| `Boss_CurseOrb` | 보스 오브젝트 (씬 위쪽 중앙) |
| `PlayerSpawn` | 플레이어 스폰 위치 (씬 아래 중앙) |
| `Canvas` | 보스 HP 바 UI |
| `GameManager` | 씬에 배치 (DontDestroyOnLoad 대상) |
