## 이 단계에서 만드는 것

04 전투에서 플레이어는 단일 공격 히트박스 하나만 가졌습니다.  
이 단계에서는 **여러 스킬을 키 하나씩 배정**하고, 스킬별로 수치·이펙트·사운드를 다르게 설정할 수 있는 구조를 기획합니다.

아울러 "게임이 어떻게 시작하고 어떻게 끝나는지"를 책임지는 **GameManager** 의 역할도 이 단계에서 먼저 정의합니다.  
코드 구현은 07-코드 문서에서 합니다.

---

## 1. 스킬이란 무엇인가

스킬 = 플레이어가 **특정 키를 눌렀을 때** 발생하는 동작의 묶음입니다.

| 구성 요소 | 설명 |
|-----------|------|
| **키 배정** | 어떤 키를 누르면 발동하는가 |
| **데미지** | 적에게 얼마를 깎는가 |
| **쿨타임** | 몇 초 후에 다시 쓸 수 있는가 |
| **사거리/범위** | 닿는 거리 또는 범위 반경 |
| **이펙트** | 어떤 시각 효과가 나타나는가 |
| **사운드** | 어떤 소리가 나는가 |

이 정보를 Unity의 **ScriptableObject(데이터 에셋)** 에 담아 두면, 스킬을 추가할 때 코드를 고치지 않고 에셋만 만들면 됩니다.

## 2. 우리 게임에 들어갈 스킬 목록

| 번호 | 스킬 이름 | 키 | 데미지 | 쿨타임 | 방식 | 비고 |
|:----:|-----------|-----|--------|--------|------|------|
| 1 | **기본 공격** | X | 10 | 0.5초 | 근접 단타 | 04에서 만든 히트박스 확장 |
| 2 | **강타** | Z | 30 | 2.0초 | 근접 단타 | 데미지·쿨만 다름 |

> 수치는 초기값입니다. 에셋을 만들고 Play 테스트 후 조정합니다.

## 3. 스킬 유형별 차이

### 단타형 (기본 공격, 강타)

- 플레이어 앞쪽 한 방향으로 레이캐스트 또는 박스 오버랩.
- 닿은 적 **1마리**에게 데미지.
- 이펙트: 적의 위치에 생성.

### 범위형 (범위 공격)

- 플레이어 중심에서 반경 안의 **모든 적**에게 데미지.
- `Physics2D.OverlapCircleAll`로 반경 안 콜라이더를 모두 찾음.
- 이펙트: 플레이어 위치에 생성 (1개).

### 이동형 (돌진)

- 짧은 시간 동안 플레이어가 앞쪽으로 빠르게 이동.
- 이동 경로상 적에게 1회 데미지.
- `Rigidbody2D`로 이동해 물리 충돌을 유지합니다.

## 4. GameManager의 역할 정의

GameManager는 게임 전체 흐름을 조율하는 스크립트입니다.  
UI나 전투 스크립트가 직접 씬을 전환하거나 게임 오버를 처리하면 나중에 코드가 얽힙니다.  
**한 군데에서만** 흐름을 결정하도록 아래 역할을 GameManager에 몰아둡니다.

| 역할 | 설명 |
|------|------|
| **게임 오버 처리** | `GameEvents.GameOver` 이벤트를 받아 UI에 알리고 `Time.timeScale = 0` |
| **재시작** | 현재 씬 다시 로드 |
| **씬 전환** | 다음 스테이지, 메인 메뉴 이동 |
| **Singleton** | 씬이 전환돼도 하나만 존재 (`DontDestroyOnLoad` 사용) |

> GameManager가 갑자기 등장하면 혼란스럽습니다.  
> 05 UI 코드에서 재시작 버튼을 만들 때 `SceneManager.LoadScene`을 직접 썼다면, 이 단계에서 GameManager를 먼저 만들고 그 메서드를 호출하도록 교체합니다.

## 5. 스킬 데이터 설계 (기획 관점)

스킬 정보를 ScriptableObject(SO)로 만드는 이유:

- **수치 조정**: 코드 수정 없이 Inspector에서 수치만 바꾸면 됨.
- **스킬 추가**: 새 SO 에셋 하나 만들고 배열에 추가하면 새 스킬 완성.
- **기획·코드 분리**: 기획 담당은 SO를 채우고, 코드 담당은 SO를 읽는 로직만 관리.

| SkillData 필드 | 타입 | 설명 |
|----------------|------|------|
| `skillName` | string | 스킬 이름 (예: 기본 공격) |
| `damage` | int | 데미지 |
| `cooldown` | float | 쿨타임 (초) |
| `range` | float | 단타 사거리 |
| `radius` | float | 범위 반경 (0이면 단타) |
| `effectPrefab` | GameObject | 발동 이펙트 프리팹 |
| `soundClip` | AudioClip | 발동 사운드 |
