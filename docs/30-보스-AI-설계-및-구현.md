# Part 1. 보스 AI 설계 (기획)

## 1.1 왜 상태 머신(FSM)인가

| 방식 | 난이도 | 보스에 적합 |
|------|--------|-------------|
| **FSM (상태 전환)** | ★★☆ | 패턴이 정해져 있을 때 **가장 추천** |
| 행동 트리 | ★★★ | 패턴이 매우 많을 때 |
| 단순 if 나열 | ★ | 소형 보스만 |

보스는 **「지금은 이 공격만 한다」** 구간이 명확하므로 **FSM**으로 구현합니다.

## 1.2 기본 상태 목록 (권장)

| 상태(영문) | 한글 | 하는 일 |
|------------|------|---------|
| `Intro` | 등장 | (선택) 무적, 연출, 카메라 |
| `Idle` | 대기 | 짧은 휴식, 다음 패턴 선택 |
| `Move` | 이동 | 플레이어 쪽 X 추적 또는 공중 부유 Y 고정 |
| `AttackA` | 공격 A | 예: 탄막(원형 발사) |
| `AttackB` | 공격 B | 예: 지면 강타(범위 데미지) |
| `AttackC` | 공격 C | 예: 소환·레이저 예고선 |
| `Stunned` | 휨/약점 | 특정 조건 시(선택) |
| `Dead` | 사망 | 체력 0, 클리어 이벤트 |

## 1.3 페이즈(체력 구간)

| 페이즈 | HP 비율 | 기획 예시 |
|--------|---------|-----------|
| 1 | 100% ~ 66% | 느린 패턴, 탄 6발 |
| 2 | 66% ~ 33% | 탄 10발 + 강타 추가 |
| 3 | 33% ~ 0% | 연속 패턴, 속도↑ |

→ **데이터(SO)**에 `hpPhase2Threshold`(0.66f), `hpPhase3Threshold`(0.33f) 로 둡니다.

## 1.4 기획서 「저주 구체」에 맞춘 패턴 예시

| 패턴 | 플레이어 경험 | 구현 요약 |
|------|---------------|-----------|
| **어둠 구체 주변 탄막** | 피격 회피 | N방향으로 `BossProjectile` 스폰 |
| **구체 돌진** | 점프/회피 | 보스 X속도 급가속 후 잠시 Idle |
| **저주 장판** | 바닥 피하기 | 바닥 Trigger, `CurseStack` 또는 데미지 |
| **핵심 노출(약점)** | 딜 타임 | 특정 상태에서만 `Collider` 약점 활성 |

# Part 2. 데이터 (ScriptableObject)

## 2.1 `BossPhaseData.cs` (한 페이즈의 공격 설정)

```csharp
using UnityEngine;

[System.Serializable]
public class BossAttackSlot
{
    public string stateName = "AttackA"; // FSM 상태 이름과 동일하게
    public float weight = 1f;            // 랜덤 선택 가중치
    public float cooldownAfter = 2f;     // 이 공격 끝난 뒤 쉬는 시간
}

[System.Serializable]
public class BossPhaseSettings
{
    [Tooltip("이 페이즈는 현재 HP비율이 이 값 이하일 때 적용. phases[0]=1페이즈(예: 1.0), [1]=0.66 이하=2페이즈, [2]=0.33 이하=3페이즈")]
    [Range(0f, 1f)] public float hpBelow = 1f;
    public BossAttackSlot[] attacks;
    public float moveSpeed = 2f;
    public int projectileCount = 6;
    public float projectileSpeed = 5f;
}
```

# Part 3. 완전 구현 코드

## 3.1 `BossHealth.cs` — 체력·페이즈 이벤트

```csharp
using UnityEngine;
using UnityEngine.Events;

public class BossHealth : MonoBehaviour
{
    [SerializeField] private BossData data;
    private int _hp;
    public int CurrentHp => _hp;
    public int MaxHp => data != null ? data.maxHp : 100;
    public UnityEvent<int> onHpChanged;
    public UnityEvent<int> onPhaseChanged; // 1,2,3
    public UnityEvent onDeath;

    private int _phaseIndex;

    private void Start()
    {
        _hp = MaxHp;
        _phaseIndex = 0;
        onHpChanged?.Invoke(_hp);
    }

    public void TakeDamage(int dmg)
    {
        if (_hp <= 0) return;
        _hp -= dmg;
        onHpChanged?.Invoke(_hp);
        int prev = _phaseIndex;
        UpdatePhaseIndex();
        if (_phaseIndex != prev) onPhaseChanged?.Invoke(_phaseIndex + 1);
        if (_hp <= 0)
        {
            _hp = 0;
            onDeath?.Invoke();
        }
    }

    /// <summary>phases[].hpBelow 내림차순(1 → 0.66 → 0.33)일 때, r이 만족하는 가장 마지막(강한) 페이즈.</summary>
    private void UpdatePhaseIndex()
    {
        if (data == null || data.phases == null || data.phases.Length == 0) return;
        float r = (float)_hp / Mathf.Max(1, MaxHp);
        _phaseIndex = 0;
        for (int i = 0; i < data.phases.Length; i++)
            if (r <= data.phases[i].hpBelow) _phaseIndex = i;
    }

    public BossPhaseSettings GetCurrentPhaseSettings()
    {
        if (data == null || data.phases == null || data.phases.Length == 0) return null;
        UpdatePhaseIndex();
        return data.phases[_phaseIndex];
    }
}
```

## 3.2 `BossProjectile.cs` — 탄막

```csharp
using UnityEngine;

public class BossProjectile : MonoBehaviour
{
    private float _speed = 5f;
    [SerializeField] private int damage = 10;
    [SerializeField] private float lifeTime = 5f;
    private Vector2 _dir;
    private LayerMask _playerLayer;

    public void Init(Vector2 direction, LayerMask playerLayer, float speed)
    {
        _dir = direction.normalized;
        _playerLayer = playerLayer;
        _speed = speed;
        Destroy(gameObject, lifeTime);
    }

    private void Update()
    {
        transform.position += (Vector3)(_dir * _speed * Time.deltaTime);
    }

    private void OnTriggerEnter2D(Collider2D o)
    {
        if (((1 << o.gameObject.layer) & _playerLayer) == 0) return;
        var ph = o.GetComponent<PlayerHealth>();
        if (ph != null) ph.TakeDamage(damage);
        Destroy(gameObject);
    }
}
```

**프리팹**: 스프라이트 + `CircleCollider2D` **Is Trigger** + `BossProjectile`.

## 3.3 `BossAI2D.cs` — FSM 본체 (복사용 전체)

```csharp
using System.Collections;
using UnityEngine;

/// <summary>2D 보스 FSM. BossData + BossHealth 필수.</summary>
[RequireComponent(typeof(BossHealth))]
public class BossAI2D : MonoBehaviour
{
    [SerializeField] private BossData data;
    [SerializeField] private Transform player;
    [SerializeField] private Transform[] firePoints; // 비우면 자기 위치에서 발사

    private BossHealth _health;
    private string _state = "Idle";
    private float _stateTimer;
    private bool _busy;

    private void Awake()
    {
        _health = GetComponent<BossHealth>();
        if (player == null && GameObject.FindGameObjectWithTag("Player") != null)
            player = GameObject.FindGameObjectWithTag("Player").transform;
    }

    private void Start()
    {
        _health.onDeath.AddListener(OnBossDead);
        StartCoroutine(MainLoop());
    }

    private void OnBossDead()
    {
        StopAllCoroutines();
        _state = "Dead";
        // 클리어 UI, 다음 씬 등
    }

    private IEnumerator MainLoop()
    {
        yield return new WaitForSeconds(0.5f);
        while (_health.CurrentHp > 0)
        {
            var phase = _health.GetCurrentPhaseSettings();
            if (phase == null || phase.attacks == null || phase.attacks.Length == 0)
            {
                yield return new WaitForSeconds(1f);
                continue;
            }

            var slot = PickAttack(phase.attacks);
            yield return RunState(slot.stateName, phase);
            yield return new WaitForSeconds(slot.cooldownAfter);
        }
    }

    private BossAttackSlot PickAttack(BossAttackSlot[] slots)
    {
        float sum = 0f;
        foreach (var s in slots) sum += s.weight;
        float r = Random.Range(0f, sum);
        foreach (var s in slots)
        {
            r -= s.weight;
            if (r <= 0f) return s;
        }
        return slots[0];
    }

    private IEnumerator RunState(string state, BossPhaseSettings phase)
    {
        _busy = true;
        switch (state)
        {
            case "AttackA":
                yield return AttackRadialBurst(phase);
                break;
            case "AttackB":
                yield return AttackSlam(phase);
                break;
            case "AttackC":
                yield return AttackChargeDash(phase);
                break;
            case "Move":
                yield return MoveTowardPlayer(phase.moveSpeed, 1.5f);
                break;
            default:
                yield return new WaitForSeconds(0.8f);
                break;
        }
        _busy = false;
    }

    private IEnumerator AttackRadialBurst(BossPhaseSettings phase)
    {
        int n = phase.projectileCount;
        float spd = phase.projectileSpeed;
        for (int i = 0; i < n; i++)
        {
            float ang = (360f / n) * i * Mathf.Deg2Rad;
            Vector2 dir = new Vector2(Mathf.Cos(ang), Mathf.Sin(ang));
            SpawnProjectile(dir, spd);
        }
        yield return new WaitForSeconds(0.3f);
    }

    private void SpawnProjectile(Vector2 dir, float speed)
    {
        if (data.projectilePrefab == null) return;
        Vector3 pos = firePoints != null && firePoints.Length > 0
            ? firePoints[Random.Range(0, firePoints.Length)].position
            : transform.position;
        var go = Instantiate(data.projectilePrefab, pos, Quaternion.identity);
        var bp = go.GetComponent<BossProjectile>();
        if (bp != null) bp.Init(dir, data.playerLayer, speed);
    }

    private IEnumerator AttackSlam(BossPhaseSettings phase)
    {
        // 바닥 범위: 자식에 SlamHitbox (Trigger) 켜기
        var hitbox = transform.Find("SlamHitbox")?.GetComponent<Collider2D>();
        if (hitbox != null) hitbox.enabled = true;
        yield return new WaitForSeconds(0.4f);
        if (hitbox != null) hitbox.enabled = false;
    }

    private IEnumerator AttackChargeDash(BossPhaseSettings phase)
    {
        if (player == null) yield break;
        float dir = Mathf.Sign(player.position.x - transform.position.x);
        float t = 0f;
        while (t < 0.6f)
        {
            transform.position += Vector3.right * dir * phase.moveSpeed * 3f * Time.deltaTime;
            t += Time.deltaTime;
            yield return null;
        }
    }

    private IEnumerator MoveTowardPlayer(float speed, float duration)
    {
        float t = 0f;
        while (t < duration && player != null)
        {
            float dx = Mathf.Sign(player.position.x - transform.position.x);
            transform.position += Vector3.right * dx * speed * Time.deltaTime;
            t += Time.deltaTime;
            yield return null;
        }
    }
}
```

**수정**: `BossProjectile.Init`는 위 Part 3.2와 맞춤. `SendMessage` 줄은 제거해도 됨.

## 3.5 보스에게 닿으면 데미지(선택)

```csharp
// BossBodyContact2D.cs — 보스 본체 Collider (플레이어와 충돌)
public class BossBodyContact2D : MonoBehaviour
{
    [SerializeField] private BossData data;
    [SerializeField] private float hitCooldown = 1f;
    private float _next;

    private void OnCollisionStay2D(Collision2D c)
    {
        if (data == null) return;
        if (((1 << c.gameObject.layer) & data.playerLayer) == 0) return;
        if (Time.time < _next) return;
        var ph = c.collider.GetComponent<PlayerHealth>();
        if (ph != null) ph.TakeDamage(data.contactDamage);
        _next = Time.time + hitCooldown;
    }
}
```

# Part 4. 에디터에서 할 일 (순서)

1. **빈 오브젝트** `Boss_CurseOrb` — 스프라이트(검은 구체 플레이스홀더).
2. **Rigidbody2D** Kinematic 또는 Static(이동만 `transform`으로 할 경우 Static).
3. **Collider2D** — 보스 히트박스(플레이어 공격은 기존 `EnemyHealth` 또는 보스 전용 `BossHealth.TakeDamage` 호출하도록 플레이어 공격 스크립트에 `BossHealth` 분기 추가).
4. **BossHealth** — `BossData` 할당.
5. **BossAI2D** — 같은 `BossData`, **Player** 태그 확인.
6. 자식 **SlamHitbox** — BoxCollider2D Trigger, `BossSlamHitbox`, 기본 비활성.
7. **Projectile 프리팹** — `BossProjectile` + Trigger + 스프라이트 → `BossData.projectilePrefab`.
8. **클리어**: `BossHealth.onDeath`에 `LoadScene`(다음 스테이지) 또는 UI 연결.
