## 1. 증강 아이콘 이론

### 아이콘의 역할

증강 카드에 표시되는 아이콘은 플레이어가 증강 효과를 **직관적으로** 파악하는 데 도움을 줍니다.  
글자만 있는 카드보다 아이콘이 있는 카드가 선택 속도를 높이고 게임 몰입도를 올립니다.

### 아이콘 규격

| 항목 | 값 |
|------|-----|
| 크기 | 64×64 px |
| 포맷 | PNG (투명 배경) |
| Pixels Per Unit | 100 (UI 전용이므로 100 사용) |
| Filter Mode | Point (픽셀아트) 또는 Bilinear (일반) |
| Mesh Type | Full Rect |

> UI 전용 스프라이트는 Pixels Per Unit을 100으로 설정합니다.  
> 게임 오브젝트 크기와 혼동하지 않도록 UI 스프라이트 폴더를 따로 관리하세요.

### 아이콘 디자인 가이드

| 증강 이름 | 아이콘 표현 아이디어 |
|-----------|---------------------|
| 강철 심장 | 하트 + 갑옷/강철 질감 |
| 신속 | 신발 또는 번개 |
| 날카로운 날 | 검 또는 칼날 |
| 빠른 손 | 손목 + 속도선 |
| 두꺼운 갑옷 | 방패 또는 갑옷 |

아이콘을 그리기 어렵다면 Unity Package Manager에서 **2D Sprite**를 사용하거나, 단색 도형으로 임시 제작합니다.

## 2. 증강 아이콘 Unity 임포트

1. `Assets/Sprites/UI/Augments/` 폴더 만들기.
2. 아이콘 PNG 드래그.
3. Texture Type: **Sprite (2D and UI)**.
4. Pixels Per Unit: **100**.
5. Mesh Type: **Full Rect**.
6. **Apply**.

## 3. AugmentData 에셋 만들기

### 전제 조건

`AugmentData.cs` 스크립트가 있어야 합니다. 10-코드 문서에서 먼저 만드세요.

## 4. 카드 배경 이미지

증강 카드 배경에 사용할 UI 이미지입니다.

| 항목 | 값 |
|------|-----|
| 크기 | 200×280 px |
| 포맷 | PNG |
| 등급별 색상 | Common: 회색, Rare: 초록, Epic: 빨강 |

등급에 따라 카드 테두리 색이 달라지면 희귀 증강을 구분하기 쉽습니다.  
단색으로 만든 후 Unity UI Image 컴포넌트의 `Color`로 등급별 색을 코드에서 설정할 수도 있습니다.

## 5. 메인 메뉴 씬 기본 구성

### 씬 생성

1. **File → New Scene** → 이름 `MainMenu`로 저장.
2. **File → Build Settings → Add Open Scenes**.

### UI 구성

1. Hierarchy → **UI → Canvas** → Scale With Screen Size (1920×1080).
2. Canvas 안에 빈 오브젝트 `MainMenuUI` 추가.
3. 자식 추가:
   - `TitleText` — TextMeshPro, 게임 타이틀 입력.
   - `StartButton` — Button, 텍스트 "게임 시작".
   - `QuitButton` — Button, 텍스트 "종료".
