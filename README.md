# Unity 2D 도트 액션 플랫포머 (팀·강좌)

**저장소에는 문서만** 있습니다. Unity Hub에서 **2D 프로젝트**를 새로 만든 뒤, **아래 단계(A→H)** 순서대로 문서를 열고 적용하면 **PC 빌드까지** 갈 수 있습니다.

**환경**: Unity 6.3 **LTS** · **PC**(Windows/Mac) — 상세 [02 §1.4](docs/02-준비사항-체크리스트.md)

**파일 팁**: `03-게임기획서-초안` 과 `03-게임기획서-리뷰` 는 **서로 다른 파일**입니다.
